     <h2 class="intitule">
      Dépannage Informatique Pour les Particuliers
     </h2>
     <p class="description">
      ORDIOUEST est une société en auto-entrepreneur pour le dépannage informatique pour les particuliers qui applique des tarifs raisonnables.
      <br/>
      <br/>
      Dans ces tarifs sont compris les déplacements pour l'intervention ou bien pour récupérer la machine. Il n'y aura aucuns dépassements d'honoraires pour le client.
      <br/>
      <br/>
      Assistance au domicile ou à l'atelier:
      <br/>
      <br/>
      Prix : 50€
      <br/>
      <br/>
      Cette formule vous permet d'obtenir l'intervention d'un technicien informatique qui viendra dépanner votre ordinateur PC (suppression virus, base de registre, vérification complète de votre système), installation d'un périphérique (installation d'une imprimante, scanner, webcam, etc...) ou prendre votre matériel et faire l'intervention à l'atelier. Ce type de formule est valable pour la plupart des pannes mineures
      <br/>
      <br/>
      Remise à zéro
      <br/>
      <br/>
      Prix : 60€
      <br/>
      Retrouvez votre ordinateur comme neuf ! Formatage du disque dur avec sauvegarde de vos données au préalable, installation ou réinstallation de Windows (xp,vista ou seven), installation des drivers et enfin réinstallation de vos logiciels puis transfert de vos données sauvegardées sur votre nouvelle configuration. Cette solution radicale est parfois nécessaire lorsqu'un ordinateur est trop infecté par des virus ou tout simplement lorsque le système est installé depuis longtemps et que vous commencez à constater des ralentissements souvent dus à une certaine usure du système d'exploitation Windows installé sur votre ordinateur.
      <br/>
      <br/>
      *************************************************************************
      <br/>
      <br/>
      NOUVEAU : Contrat d'entretien annuel pour vos PC : PRIX : 90 annuel
      <br/>
      Une révision complète par an + une intervention à 50€ ou 60€ (voir ci-dessus) gratuite.
      <br/>
      Pour plus de renseignement merci de prendre contact avec moi.
      <br/>
      <br/>
      ************************************************************************
      <br/>
      <br/>
      Assemblage personnalisé de votre PC selon votre budget et vos besoins.
      <br/>
      <br/>
      Intervention dans un rayon de 50kms à partir de Nort sur Erdre.
      <br/>
      Les devis sont gratuits.
      <br/>
      <br/>
      Pour la prise d'un rendez-vous merci de me joindre du lundi au samedi de 9h à 19h:
      <br/>
      <br/>
      Par téléphone, Mail ou SMS: 06/68/30/17/15
      <br/>
      <br/>
      Pour les tarifs :
      <br/>
      <br/>
      les tarifs sont pour une intervention et non de l'heure (Ex: 1h ou 4H d'intervention=50€).
      <br/>
      <br/>
      Ordiouest
