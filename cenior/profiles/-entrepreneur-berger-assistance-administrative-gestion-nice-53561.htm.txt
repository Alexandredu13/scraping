     <h2 class="intitule">
      Assistance administrative et de gestion
     </h2>
     <p class="description">
      Assistante administrative et de gestion polyvalente indépendante à Nice vous propose ses prestations de services, vous vous trouvez dans l'une de ces situations :
      <br/>
      <br/>
      - vous souhaitez déléguer vos tâches administratives
      <br/>
      - vous faites face à une surcharge d’activité ponctuelle ou récurrente
      <br/>
      - vous êtes en démarrage d’activité
      <br/>
      - votre secrétaire est absente
      <br/>
      <br/>
      Je suis à votre disposition pour vous aider dans vos tâches quotidiennes
      <br/>
      <br/>
      Voici une liste non exhaustive des activités pouvant m'être confiées :
      <br/>
      <br/>
      ASSISTANCE ADMINISTRATIVE :
      <br/>
      - Accueil physique / téléphonique
      <br/>
      - Saisie et mise en page de documents
      <br/>
      - Retranscription de fichiers audio
      <br/>
      - Création / mise à jour base de données
      <br/>
      - Mailings
      <br/>
      - Tri / Classement / Archivage
      <br/>
      - Numérisation / Photocopies de vos documents
      <br/>
      - Mise sous plis
      <br/>
      - Gestion de votre agenda / de votre messagerie
      <br/>
      <br/>
      ASSISTANCE DE GESTION :
      <br/>
      - Règlements clients et fournisseurs
      <br/>
      - Suivi créances clients
      <br/>
      - Relances des impayés
      <br/>
      - Analyse grand livre clients
      <br/>
      - Réalisation de tableaux de gestion (suivi de trésorerie, de CA...)
      <br/>
      - Facturation
      <br/>
      - Devis
      <br/>
      <br/>
      Je suis polyvalente, méthodique et rigoureuse, je bénéficie d'une expérience significative de 10 ans en tant qu'assistante de gestion interne au sein d'un cabinet d'expertise comptable, d'une société d’électricité générale et d'une enseigne de prêt à porter.
      <br/>
      <br/>
      Je maîtrise parfaitement l'outil informatique, et je suis complètement équipée pour vous assister à distance.
      <br/>
      <br/>
      Chaque demande de services est unique contactez-moi pour obtenir un devis gratuit détaillé.
      <br/>
      <br/>
      Je suis à votre disposition pour répondre à vos besoins et à vos attentes
      <br/>
      <br/>
      Tél. : 06.60.54.28.69
      <br/>
      Mail : azuractiv.gestion@gmail.com
      <br/>
      <br/>
      Prestations à distance ou sur site
      <br/>
      <br/>
      Du lundi au vendredi :
      <br/>
      de 9H à 12H - et de 13H à 18H
      <br/>
     </p>
