     <h2 class="intitule">
      Travaux et d'installation et de maitrise d'oeuvre dans le chauffage, ventilation, climatisation, plomberie, pompe à chaleur, suivi de chantier, chiffrage et devis
     </h2>
     <p class="description">
      Depuis mars 2011 j'ai crée une structure permettant au société entre 0 et 50 salarié de pouvoir externaliser les taches liés à la maitrise d'œuvre dans le génie climatique :
      <br/>
      <br/>
      • chauffage,
      <br/>
      • ventilation,
      <br/>
      • climatisation (détente direct et eau glacée).
      <br/>
      • plomberie,
      <br/>
      • traitement d'air,
      <br/>
      • solaire thermique,
      <br/>
      • pompe à chaleur,
      <br/>
      • système ENR
      <br/>
      <br/>
      › Nos prestations :
      <br/>
      <br/>
      • Conseil technique
      <br/>
      • Etude de prix
      <br/>
      • Etude d'exécution
      <br/>
      • Plan et schéma d'installation CVC / PB
      <br/>
      • Note de calculs (dimensionnement d'installation, sélection de matériel, ...)
      <br/>
      • Dossiers techniques
      <br/>
      • Conduite de travaux
      <br/>
      • Audit technique
      <br/>
      • Suivi de contrat de maintenance d'installation thermique
      <br/>
      <br/>
      › Nos domaines d'activités :
      <br/>
      <br/>
      • Maison individuelle
      <br/>
      • Logement collectif
      <br/>
      • Hotels
      <br/>
      • Installation technique (chaufferie, local technique)
      <br/>
      • Salle de réunion
      <br/>
      • Lieux de culte
      <br/>
      <br/>
      › Nos secteurs géographique :
      <br/>
      <br/>
      - Ile de france
      <br/>
      - Région parisienne
      <br/>
      - Oise
      <br/>
      - Eure
      <br/>
      - Calvados
      <br/>
      - Tarn et garonne
      <br/>
      <br/>
      Secteur : Ingénierie - Bureau d'Études - Conseil
     </p>
