     <h2 class="intitule">
      tous travaux de rénovation
     </h2>
     <p class="description">
      Je suis à votre  écoute
      <br/>
      pour tous vos besoins concernant la plupart des travaux que vous souhaitez réaliser aussi bien pour embellir que pour agrandir votre logement.
      <br/>
      <br/>
      Je me déplace gratuitement afin de voir avec vous toutes les possibilités pour la réalisation de vos projets et une estimation des travaux.
      <br/>
      <br/>
      Je suis aussi à même de réaliser pour vous toutes les démarches concernant une demande de permis de construire ou une simple autorisation de travaux (exécution des plan et de tous documents à déposer en mairie).
      <br/>
      <br/>
      Je m’engage a vous transmettre un devis dans les plus brefs délais.
      <br/>
      <br/>
      Mes 40 années passées dans le domaine du bâtiment tous corps d’états mon permis d’obtenir un très bon acquis des connaissances pour la réalisation de tous ces travaux même ceux les plus complexe.
      <br/>
      <br/>
      Voici quelques possibilités de réalisation que je peux réaliser (Liste non exhaustive).
      <br/>
      <br/>
      Aménagement de combles.
      <br/>
      <br/>
      Création de piéces
      <br/>
      <br/>
      Aménagement de salle de bains
      <br/>
      <br/>
      Aménagement de cuisine ( meubles et raccordement)
      <br/>
      <br/>
      Touts travaux de maçonnerie, de placo-platre.
     </p>
