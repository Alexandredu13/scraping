     <h2 class="intitule">
      RENOVATION de l HABITAT ELECTRICIEN GENERALISTE TOUS LOCAUX
     </h2>
     <p class="description">
      Mesdames, messieurs Bonjour.
      <br/>
      <br/>
      Vous cherchez un électricien professionnel sur le secteur de Sainte-Foy-la-Grande / Bergerac / Libourne et CUB ?
      <br/>
      <br/>
      J'étudie tous vos projets de travaux électriques d'installations neuves,
      <br/>
      en rénovation / mise aux normes / extension ou dépannage.
      <br/>
      <br/>
      Je suis à l'écoute de vos attentes, vos préférences dans le soucis de vous conseiller au mieux sur le matériel nécessaire à votre projet et les travaux à prévoir pour une installation aux normes.
      <br/>
      <br/>
      <br/>
      AVANTAGES DE MES PRESTATIONS DE SERVICES:
      <br/>
      <br/>
      *Devis gratuit sur demande.
      <br/>
      *Tarifs très attractifs de 20 à 25 euros de l'heure, 25 à 30 euros en dépannage (Supplément le soir après 18h et le week end).
      <br/>
      *Déplacement sans supplément sur le secteur Foyen et Bergerac.
      <br/>
      *Tarifs sans TVA facturable pour la main d'oeuvre.
      <br/>
      (Article 293B du CGI).
      <br/>
      *Matériels et fournitures électriques aux tarifs réduits "professionnels" sans aucun frais supplémentaires.
      <br/>
      *Travaux et installations garantis 10 ans.
      <br/>
      <br/>
      CONDITIONS DE MES PRESTATIONS DE SERVICES:
      <br/>
      <br/>
      *50% d'acompte à la signature du devis.
      <br/>
      *Aucune avance de matériel sans acompte.
      <br/>
      *Supplément à partir de 20e pour déplacement de plus de 30 kms.
      <br/>
      <br/>
      Pour tous renseignements ou prise de contact pour étude de votre projet et devis gratuit, n’hésitez pas!
      <br/>
      <br/>
      Je suis joignable au: 07 64 09 91 25 / 09 81 41 61 17.
      <br/>
      <br/>
      Électriquement vôtre!
