     <h2 class="intitule">
      lavage auto sans eau a domicile
     </h2>
     <p class="description">
      <br/>
      <br/>
      Eco'Clean
      <br/>
      Lavage auto et moto
      <br/>
      <br/>
      Eco'Clean est un concept de nettoyage et de protection de vehicule à domicile.
      <br/>
      ... Votre vehicule a besoin d'une beauté?
      <br/>
      <br/>
      N'hesitez pas à nous contacter 7j/7 , nous nous deplacons ou vous le souhaitez (domicile,bureau,etc..) dans un rayon de 40kms (sans frais supplémentaires), c'est à dire 40kms autour de bazoches les gallerandes et 40 kms autour d'orleans.
      <br/>
      <br/>
      TARIFS
      <br/>
      <br/>
      # Nettoyage simple:
      <br/>
      <br/>
      - interieur: dépoussierage des plastiques , aspiration complete du vehicule , nettoyage vitres et seuil de porte.
      <br/>
      25e
      <br/>
      <br/>
      - exterieur: lavage complet carosserie , nettoyage jantes (enjoliveurs) et vitres.
      <br/>
      25e
      <br/>
      <br/>
      # Formule interieur + exterieur : 40e
      <br/>
      <br/>
      # prestations supplementaires:
      <br/>
      <br/>
      -interieur:
      <br/>
      - shampoing sieges avant+arriere 30e
      <br/>
      - shampoing moquette et tapis 15e
      <br/>
      - entretien et renovation cuir 45e
      <br/>
      - detachant sieges , moquette 15e
      <br/>
      - nettoyage poils d'animaux sur devis
      <br/>
      <br/>
      -exterieur:
      <br/>
      - degoudronant 10e
      <br/>
      - brillant pneus 7e
      <br/>
      - soin chrome 10e
      <br/>
      - efface rayures 10e
      <br/>
      - renov plastiques ext 15e
      <br/>
      - renov/polish 30e
      <br/>
      - bloc moteur 10e
      <br/>
      <br/>
      <br/>
      lavage moto sans chrome 20e
      <br/>
      lavage moto avec chrome 35e
      <br/>
      <br/>
      Autres formules sur simple demande (formules a la carte)
      <br/>
      <br/>
      <br/>
      vous pouvez nous contacter a toute heure.
      <br/>
      Erwan 06/19/66/29/74
      <br/>
      Morgan 07/78/54/26/57
      <br/>
      <br/>
      ecocleanauto45@gmail.com
