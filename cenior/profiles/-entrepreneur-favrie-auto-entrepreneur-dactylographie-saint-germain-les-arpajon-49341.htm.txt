     <h2 class="intitule">
      Auto entrepreneur en dactylographie de documents
     </h2>
     <p class="description">
      <br/>
      Auto-entrepreneur déclarée - SIRET 752 280 545 00019 - et travaillant dans le milieu du Secrétariat depuis plus de 20 ans, je vous garantie un travail confidentiel, rapide et soigné.
      <br/>
      <br/>
      Vous êtes situé en France, vous n’avez pas de secrétaire, vous manquez de temps, alors n’hésitez pas et confiez-moi vos travaux de dactylographie.
      <br/>
      <br/>
      Vous êtes Artisan, une Société ou un particulier, vous avez besoin d’une personne pour dactylographier vos documents (devis, factures, retranscription de tous types de documents, courriers, relances…) vous me transmettez vos brouillons scannés par mail ou par courrier, je les dactylographie et les mets en page puis vous transmets vos documents dactylographiés (mail, courrier ou sur clé USB selon votre choix).
      <br/>
      <br/>
      Mes tarifs sont sans TVA selon la loi en vigueur.
      <br/>
      <br/>
      Le travail effectué :
      <br/>
      • Saisie de devis, factures, courriers : 3.50 € la page
      <br/>
      • Saisie de CV, tableaux : 4.50 € la page
      <br/>
      • Saisie de roman, biographie, thèse, mémoire : sur devis
      <br/>
      La saisie comprend la mise en page et la correction orthographique
      <br/>
      • Relecture et correction orthographique de documents : 9.00 € l'heure
      <br/>
      <br/>
      Les impressions de vos documents :
      <br/>
      • Impression A4 noir et blanc : 0.20 € la page
      <br/>
      • Impression A4 couleur : 0.40 € la page
      <br/>
      <br/>
      L’envoi de vos documents :
      <br/>
      • Envoi par mail : GRATUIT
      <br/>
      • Envoi par courrier prioritaire poids jusqu’à 20 g : 0.70 €
      <br/>
      Délai d’acheminement de courrier prioritaire : J+1
      <br/>
      L’envoi des documents comprend l’enveloppe + le coût de l’affranchissement postal
      <br/>
      • Sauvegarde sur clé USB Verbatim 2 Go : 5 €
      <br/>
      <br/>
      Possibilités de forfait : exemple : 10 devis + 3 factures : 40€ (au lieu de 45.50€)
      <br/>
      <br/>
      Acceptation de la commande :
      <br/>
      Je vous enverrais le prix pour la prestation demandée par e-mail, vous m’envoyez votre accord par e-mail et le règlement par chèque, espèces ou virement bancaire.
      <br/>
      <br/>
      Le règlement est payable 100% à la commande.
      <br/>
      <br/>
      Pour toute demande ou forfait, surtout n’hésitez pas à me contacter par mail à l’adresse suivante favrie_dactylographie@ymail.com. Je répondrais à toutes vos demandes en fin de journée.
      <br/>
      <br/>
      Merci d’avoir pris le temps de lire mon annonce.
     </p>
