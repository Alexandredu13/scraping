     <h2 class="intitule">
      Coaching scolaire et professionnel
     </h2>
     <p class="description">
      Vous êtes :
      <br/>
      Des collégiens,lycéens, étudiants et déscolarisés
      <br/>
      De futurs apprentis/stagiaires en recherche
      <br/>
      d’entreprises
      <br/>
      Des adultes ou des parents dépassés et en rupture avec vos enfants
      <br/>
      Des personnes désireuses de changement.
      <br/>
      <br/>
      Les différents types d’accompagnement:
      <br/>
      <br/>
      <br/>
      <br/>
      ORIENTATION
      <br/>
      <br/>
      Aider le jeune et l’adulte à faire le choix d’orientation scolaire et professionnelle.
      <br/>
      <br/>
      <br/>
      MOTIVATION/CONFIANCE EN SOI
      <br/>
      <br/>
      Aider au changement de comportement.
      <br/>
      Créer les conditions d'une meilleure motivation.
      <br/>
      Accompagner dans une logique d'amélioration
      <br/>
      et d'épanouissement.
      <br/>
      Apprendre à gérer le stress en gagnant en
      <br/>
      confiance en soi.
      <br/>
      <br/>
      <br/>
      METHODOLOGIE
      <br/>
      <br/>
      Aider l’élève dans l’organisation de son travail.
      <br/>
      Mettre en place des outils méthodologiques
      <br/>
      permettant d’optimiser ses résultats scolaires.
      <br/>
      Développer des stratégies d’apprentissage
      <br/>
      efficaces et performantes.
      <br/>
      <br/>
      <br/>
      RECHERCHE D’ENTREPRISE
      <br/>
      <br/>
      Se préparer aux techniques de recherche d’entreprise (stage, contrat en alternance et premier emploi)
      <br/>
      <br/>
     </p>
